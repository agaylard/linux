Sunplus SP7021 Pin Controller
-----------------------------

The Sunplus SP7021 pin controller is used to set up the function of a pin or a
pin-group.

Please refer to ../gpio/gpio.txt and ../interrupt-controller/interrupts.txt for
a general description of GPIO and interrupt bindings.

Please refer to pinctrl-bindings.txt in this directory for details of the
common pinctrl bindings used by client devices.

The device node of Sunplus SP7021 pin controller has following properties.

Required properties:
 - compatible: should be "sunplus,sp7021-pctl".
 - reg: addresses and length of the register sets for pin controller.
 - reg: base address and length of the pin controller.
 - gpio-controller: mark the device node as a gpio controller.
 - #gpio-cells: should be two.
 - clocks: clock selector for pin controller.
 - resets: reset controller selector for pin controller.

Each subnode lists and configures the pins.

	subnode {
		sppctl,groups = "GROPU_NAME";
		sppctl,function = "FUNCTION_NAME";
		sppctl,pins = < PIN_NUMBER_PINMUX >;
	};

Required properties in subnode:
 - sppctl,groups: name of pin-group.
 - sppctl,function: name of pin-function.
 - sppctl,pins: pin sets, please use macro SPPCTL_IOPAD to define each pin.

Example:

#include <dt-bindings/reset/sp-q628.h>
#include <dt-bindings/clock/sp-q628.h>
#include <dt-bindings/gpio/gpio.h>
#include <dt-bindings/pinctrl/sppctl-sp7021.h>

pctl: pctl@0x9C000100 {
	compatible = "sunplus,sp7021-pctl";
	reg = <0x9C000100 0x100>, <0x9C000300 0x80>, <0x9C000380 0x80>,
	      <0x9C0032e4 0x1C>, <0x9C000080 0x20>;
	gpio-controller;
	#gpio-cells = <2>;
	clocks = <&clkc GPIO>;
	resets = <&rstc RST_GPIO>;

	pins_uart0: pins_uart0 {
		sppctl,groups = "UA0";
		sppctl,function = "UA0";
	};

	emmc_mux: emmc_mux {
		sppctl,groups = "CARD0_EMMC";
		sppctl,function = "CARD0_EMMC";
	};

	mmc1_mux: mmc1_mux {
		sppctl,groups = "SD_CARD";
		sppctl,function = "SD_CARD";
		sppctl,pins = < SPPCTL_IOPAD(91,SPPCTL_PCTL_G_GPIO,0,0) >;
	};

	hdmi_A_tx1: hdmi_A_tx1_pins {
		sppctl,groups = "HDMI_TX1";
		sppctl,function = "HDMI_TX";
	};
	hdmi_A_tx2: hdmi_A_tx2_pins {
		sppctl,groups = "HDMI_TX2";
		sppctl,function = "HDMI_TX";
	};
	hdmi_A_tx3: hdmi_A_tx3_pins {
		sppctl,groups = "HDMI_TX3";
		sppctl,function = "HDMI_TX";
	};
};
