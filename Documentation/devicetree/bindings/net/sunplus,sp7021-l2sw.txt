Sunplus SP7021 dual 10M/100M Ethernet controller (with Layer 2 switch)
----------------------------------------------------------------------

The device node of Sunplus SP7021 Ethernet (L2SW) controller has following
properties.

Required properties:
 - compatible: should be "sunplus,sp7021-l2sw".
 - reg: addresses and length of the register sets for Ethernet controller.
 - reg-names: one name for each entry in the reg property.
 - interrupt-parent: phandle for interrupt parent.
 - interrupts: number and type of interrupt, type should be high-level.
 - clocks: clock selector for Ethernet controller.
 - resets: reset controller selector for Ethernet controller.
 - phy-handle1: handle to node of phy 1 in mdio node.
 - phy-handle2: handle to node of phy 2 in mdio node.
 - pinctrl-names: Names corresponding to the numbered pinctrl states.
 - pinctrl-0: phandle for the 'default' state of pin configuration.
 - nvmem-cell-names: Names corresponding to the nvmem cells of MAC address.
 - nvmem-cells: nvmem cell address of MAC address.
 - mode:
   - 0: daisy-chain mode
   - 1: dual NIC mode
   - 2: daisy-chain mode but disable SA learning

Example:

l2sw: l2sw@0x9c108000 {
	compatible = "sunplus,sp7021-l2sw";
	reg = <0x9c108000 0x400>, <0x9c000280 0x80>;
	reg-names = "l2sw", "moon5";
	interrupt-parent = <&intc>;
	interrupts = <66 IRQ_TYPE_LEVEL_HIGH>;
	clocks = <&clkc L2SW>;
	resets = <&rstc RST_L2SW>;
	phy-handle1 = <&eth_phy0>;
	phy-handle2 = <&eth_phy1>;
	pinctrl-names = "default";
	pinctrl-0 = <&l2sw_demo_board_v3_pins>;
	nvmem-cells = <&mac_addr0>, <&mac_addr1>;
	nvmem-cell-names = "mac_addr0", "mac_addr1";
	mode = < 1 >;
};
