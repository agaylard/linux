Sunplus SoC reset controller

Required properties:
 - compatible: Must contain one of
   - "sunplus,sp7021-reset"
   - "sunplus,q645-reset"
 - #reset-cells: Specifies the number of cells needed to encode reset
                 line, should be 1.
 - reg: address and length of the register set for reset controller
 - reg-names: names for reg property

Example:
rstc: reset@0x9c000054 {
	compatible = "sunplus,sp7021-reset";
	#reset-cells = <1>;
	reg = <0x9c000054 0x28>;
	reg-names = "reset";
};
